
# Web-based Patient Data Management System with Blockchain Security

This project aims to develop a web-based system for secure patient data management, utilizing blockchain technology to ensure data integrity, security, and privacy. The system allows healthcare providers to store, manage, and access patient records in a decentralized and secure manner, preventing unauthorized access and ensuring compliance with privacy regulations.

## Features
- Secure patient data storage using blockchain
- User authentication and access control
- Decentralized data management

## Technologies Used
- Blockchain (for data security)
- Web development technologies (HTML, CSS, JavaScript, etc.)
- Backend technologies (Node.js, Express, etc.)

## Setup and Installation
1. Clone the repository:  
   `git clone https://github.com/Sheshaadhri14/Software_project.git`
2. Install dependencies:  
   `npm install`
3. Run the application:  
   `npm start`
